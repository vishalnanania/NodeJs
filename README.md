Debugging:

1) node --inspect app.js read --id=1
2) n - next line in debugger
3) list(10) - 10 lines in debugger
4) c - continue
5) debugger - put break point 
6) testing in chrome: node --inspect-brk app.js read --id=1
7) chrome://inspect
